Number of distinct substrings in a string Open on JetBrains Academy

Given a string \( s \). Write a program that counts the number of distinct substrings contained in \( s \).

NB: Remember about the empty string.

Hint: Suppose we already know the number of distinct substrings for \( s[(i+1):|s|] \). Now let's add a symbol \( s[i] \) to the beginning of this substring. Thus, we add \( \left| s[i:|s|] \right| \) new substrings. But how many of them did not appear before?


Sample Input:
aba

Sample Output:
6


Sample Input:
x

Sample Output:
2


Memory limit: 256 MB
Time limit: 5 seconds
