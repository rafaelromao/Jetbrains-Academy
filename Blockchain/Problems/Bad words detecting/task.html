Bad words detecting Open on JetBrains Academy

You need to implement a method that returns a prepared stream for detecting bad words. The method has two parameters:

  • a text in which all words are divided by single whitespaces;
  • a list of bad words.

After calling collect(Collectors.toList()) the prepared stream must return the list of bad words found in the text. The result should be dictionary ordered (i.e. sorted lexicographically) and bad words shouldn't repeat.

You need return a prepared stream from the template method, not a list of bad words. Pay attention to the method template. Do not change it!
