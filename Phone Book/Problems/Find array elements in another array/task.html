Find array elements in another array Open on JetBrains Academy

The first line contains integer \( 1 \le n \le 10^5 \). The second one contains of an array \( A[1 \ldots n] \) of \( n \) various natural numbers, not exceeding \( 10^9 \), in ascending order.

The third line contains integer \( 1 \le k \le 10^5 \). The fourth one consists of \( k \) natural numbers \( b_1, \ldots, b_k \), not exceeding \( 10^9 \).

For each \( i \) from 1 to \( k \) it is necessary to output index \( 1 \le j \le n \), for which \( A[j]=b_i \), or \( -1 \), if there is no such \( j \).


Sample Input:
5
1 5 8 12 13
5
8 1 23 1 11
Sample Output:
3 1 -1 1 -1


Memory limit: 256 MB
Time limit: 2 seconds
